# PowerShell 101

Bienvenido a **PowerShell 101**, una guía práctica para aprender PowerShell desde cero.

Este repositorio está pensado para cualquier persona que quiera comenzar a automatizar tareas, aprender scripting y entender cómo PowerShell puede ayudar en Windows, macOS y Linux.

## ¿Para quién es este repositorio?

Este material es ideal para:

- Personas que nunca han usado PowerShell
- Administradores de sistemas que quieren automatizar tareas
- Profesionales de TI que desean aprender scripting paso a paso
- Cualquier persona curiosa por la automatización

## ¿Qué vas a aprender?

Aquí encontrarás contenido para aprender:

- Qué es PowerShell y por qué usarlo
- Cómo instalar PowerShell Core
- Primeros comandos básicos
- Variables, objetos y pipeline
- Cómo crear y ejecutar scripts
- Buenas prácticas para empezar correctamente

## ¿Por qué PowerShell?

PowerShell es una herramienta moderna de automatización que permite administrar sistemas, consultar información, procesar datos y ejecutar tareas repetitivas de forma más rápida y consistente.

Entre sus ventajas están:

- Multiplataforma
- Código abierto
- Actualizaciones activas
- Excelente integración con tecnologías modernas
- Muy útil para automatización y administración

## Requisitos

- Ganas de aprender
- PowerShell instalado

## Recomendación

No intentes memorizar todo. Empieza por entender conceptos básicos y practicar comandos pequeños todos los días.

## Objetivo

La meta de este repositorio no es solo enseñar comandos, sino ayudarte a pensar en PowerShell como una herramienta para resolver problemas reales.
