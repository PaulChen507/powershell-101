# PowerShell 101

Bienvenido a **PowerShell 101**, un repositorio pensado para aprender **PowerShell desde cero** de forma práctica, clara y progresiva.

Este proyecto está dirigido a personas que quieren comenzar a automatizar tareas, entender los fundamentos de PowerShell y ver ejemplos reales aplicados a tecnologías ampliamente usadas en el mundo de TI.

---

## ¿Para quién es este repositorio?

Este material está pensado para:

- personas que nunca han usado PowerShell
- administradores de sistemas que quieren empezar a automatizar tareas
- profesionales de TI que desean aprender scripting paso a paso
- estudiantes o curiosos que quieren entender cómo funciona PowerShell
- personas que trabajan con Active Directory, Azure o Microsoft 365 y quieren empezar a usar PowerShell con más confianza

---

## ¿Qué encontrarás aquí?

Este repositorio combina tres cosas:

- **documentación guiada** para aprender los fundamentos
- **artículos de blog** para entender el valor de PowerShell y sus usos
- **scripts de ejemplo** anonimizados para practicar escenarios reales

La idea es que no solo aprendas comandos, sino también una forma de pensar la automatización.
